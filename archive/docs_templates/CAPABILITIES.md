# Capabilities (Exhaustive Draft)

This list defines what the project can do. Trim as needed.

## Core
- Load and validate local state.
- Generate plans with rationale and time estimates.
- Explain dependencies and unlock paths.
- Provide ratings with explainable factors.
- Support multiple modes or profiles.

## Planning
- Short/mid/long horizon planning.
- Alternative paths (fast/cheap/safe).
- Bottleneck detection and top blockers.

## Data
- Versioned schema and migration.
- Reference datasets for dependencies and unlocks.
- Import/export snapshots.

## UX
- CLI for fast iteration.
- GUI for dashboards and notes.
- Optional plugin/overlay integration.

## Analytics
- Compare snapshots over time.
- Estimate time-to-goal and ROI.
